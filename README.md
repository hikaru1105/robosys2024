# binaryコマンド

## 概要
 
- このコマンドは10進数で入力された整数を2進数の整数に変換して出力するものです。
- 標準入力でのコマンドの実行ができます。

## 使い方

### 準備
   
   1. リポジトリをクローンする
   ```
   $ git clone https://github.com/hikaru1105/robosys2024.git
   ```
   2. クローンしたディレクトリに移動する
   ```
   $ cd robosys2024
   ```
### 実行方法

   1. 実行方法の例
   ```
   $ echo 100 | ./binary
   ```
   2. 実行結果
   ```
   $ 1100100
   ```
